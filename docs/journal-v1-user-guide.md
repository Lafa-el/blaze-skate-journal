# Blaze Skate Journal V1.0 用户使用说明

Blaze Skate Journal 是给 Lindsay、家长和教练一起使用的滑冰训练记录工具。它帮助你记录每天训练后的感受、训练内容、教练反馈、身体状态、成绩进步和一周复盘。

建议每天训练后花 2-5 分钟记录。记录越简单、越稳定，后面复盘越有用。

## 每天怎么用

1. 打开 Daily，记录今天是不是训练日、比赛日、恢复日或休息日。
2. 填写今天的精力、酸痛、训练重点、做得好的地方、需要改进的地方。
3. 如果教练或家长有提醒，可以写在 Coach Feedback 或 Parent Note。
4. 训练完成后可以标记 Completed。

日期请使用 `YYYY-MM-DD`，例如 `2026-06-25`。

## 一次训练怎么记录 Sessions

Sessions 用来记录一次具体训练。

可以记录：

- 日期
- 训练类型：冰上、陆上、私教课、比赛、恢复
- 地点
- 时长
- 强度
- 教练
- 技术重点
- 主训练组
- 起跑、弯道、直道、接力、比赛模拟
- 圈速、装备、恢复备注

不需要每个字段都填。先记录最重要的内容：日期、类型、时长、重点、备注。

## 怎么看 Calendar

Calendar 用来回看历史。

- 有 Daily 的日期会有标记。
- 有 Training Session 的日期会有标记。
- 点击某一天，可以看到当天是否完成 Daily、当天有几次训练、总时长和主要重点。

家长可以用 Calendar 快速检查最近有没有漏记。

## 怎么做 Weekly Review

Weekly Review 是一周复盘。

可以自动生成：

- 最近 7 天训练天数
- 训练次数
- 总训练时长
- 冰上、陆上、比赛、恢复分布
- 平均精力
- 酸痛摘要
- 主要技术重点
- 缺少 Daily 的日期

然后手动补充：

- 本周最好的一刻
- 下周重点
- 家长总结

建议每周末由 Lindsay 和家长一起完成。

## Coach Notes 怎么用

Coach Notes 用来记录教练的关键反馈。

适合记录：

- 教练今天重点提醒了什么
- 哪些技术问题需要跟进
- 明天或下次训练要注意什么
- 是否关联某一次训练

教练可以直接看 Coach Notes 和 Weekly Review，快速了解这一周的重点问题。

## Performance 怎么记录 PB / 比赛成绩

Performance 用来记录成绩和 PB。

可以记录：

- 日期
- 项目，例如 500m、777m、1000m、1500m
- 用时
- 场景：训练测试、比赛、表演
- 备注

如果某个项目的时间是目前最好成绩，系统会标记为 PB。

## Body 怎么记录身体状态

Body 用来记录训练后的身体情况。

可以记录：

- 日期
- 睡眠小时数
- 疲劳等级
- 心情
- 酸痛部位
- 体重
- 身高
- 受伤或疼痛备注

如果有疼痛或不舒服，建议当天就写清楚，方便家长和教练判断是否需要调整训练。

## Videos 怎么用

Videos 目前是视频引用/链接记录，不是视频上传系统。

可以记录：

- 视频标题
- 文件名
- 外部链接
- 关联训练 ID
- 技术标签
- 分析状态
- 备注

适合保存 YouTube、云盘或其他外部视频链接，方便之后回看技术动作。

## Summer Camp 怎么用

Summer Camp 用来记录集训阶段。

可以设置集训开始和结束日期，然后查看：

- 集训天数
- 训练天数
- 训练次数
- 总训练时长
- PB 变化
- 高频技术问题
- 里程碑

集训期间建议每天记录 Daily 和 Sessions，这样 Summer Camp 页面才会更有价值。

## Export 怎么导出记录

Export 可以导出 JSON 或 CSV。

可以选择：

- 全部数据
- 自定义日期范围
- Summer Camp 范围
- 单周数据

导出适合做备份，或者给教练、家长做更详细分析。

## Settings / Language

Settings 可以：

- 切换语言
- 编辑资料
- 设置通知偏好
- 导出数据
- 进入隐私与安全
- 查看帮助
- 退出登录

语言支持 English 和 中文。

## 家长怎么检查

家长可以重点看：

- Dashboard：最近 7 天有没有训练记录，今天有没有 Daily。
- Calendar：有没有漏记。
- Weekly Review：一周训练量、酸痛、重点问题。
- Body：疲劳、睡眠、疼痛。
- Export：需要备份或分享时导出。

## 教练怎么阅读

教练可以重点看：

- Sessions：每次训练内容。
- Coach Notes：技术反馈和跟进点。
- Performance：成绩和 PB。
- Weekly Review：一周总结和下周重点。
- Videos：技术视频链接。

## 已知限制

- Videos 目前只保存视频引用或外部链接，不负责上传和存储视频文件。
- V1.0 使用 `athleteId == user.uid`，还没有家长/教练/多个运动员的独立权限模型。
- 目前没有真实 Firebase E2E 自动测试。
- Export 是下载当前账号可读数据，不是跨账号共享系统。
- PWA 可以安装，但离线体验仍依赖浏览器和缓存状态。
